# GET Markets

Retrieve list of active markets from GoPreDix.

## Endpoint

```
GET /api/v1/markets
```

## Parameters

| Name | Type | Required | Description |
|------|------|----------|---|
| page | integer | No | Page number (default: 1) |
| limit | integer | No | Results per page (default: 10, max: 100) |
| category | string | No | Filter by category (finance, sports, politics, tech, other) |
| status | string | No | Filter by status (active, locked, resolved) |
| sort | string | No | Sort order (volume, deadline, new) |
| search | string | No | Search markets by question |

## Example Usage

```javascript
const markets = await fetch('/api/v1/markets?category=finance&status=active&limit=20');
const data = await markets.json();
console.log(data.data);
```

| Code | Message |
|------|---------|
| 200 | Success |
