# Welcome to GoPreDix

GoPreDix is a **decentralized prediction market platform** built on blockchain technology. Create markets, trade predictions, and earn rewards on real-world events.

## What is GoPreDix?

GoPreDix enables you to:

- **Create Markets** - Launch prediction markets on any event
- **Trade Predictions** - Buy/sell shares of outcomes
- **Earn Rewards** - Profit from accurate predictions
- **Own Your Data** - All data stored on blockchain

## Key Features

✅ **Decentralized** - No central authority, truly owned by users
✅ **Transparent** - All trades visible on blockchain
✅ **Fast** - Transactions settle in seconds on BSC
✅ **Low Fees** - 0.5% trading fees, 1% market creation
✅ **Non-Custodial** - You control your funds with MetaMask

### How It Works

```
                        ┌──────────────────────────────┐
                        │        FRONTEND (Next.js)     │
                        │──────────────────────────────│
                        │ MetaMask Connect             │
                        │ Create / Trade Markets       │
                        │ Resolve / Dispute UI         │
                        └───────────────┬──────────────┘
                                        │ JSON-RPC (wagmi)
                                        ▼
┌──────────────────────────────────────────────────────────────────┐
│                     SMART CONTRACT (EVM)                         │
│               PredictionMarketWithMultipliers                    │
│──────────────────────────────────────────────────────────────────│
│ Markets: title, category, status, deadline, lockTime, result     │
│ Pools: totalYes, totalNo                                         │
│ Multipliers: multiplierYES, multiplierNO (×1000)                 │
│ User Bets: userBets[user][marketId][YES/NO]                      │
│                                                                  │
│  FLOW:                                                           │
│   createMarket()  → Created                                      │
│   startTrading()  → Trading                                      │
│   placeBet()      → Updates pools + multipliers                  │
│   resolveMarket() → Resolved + PnL                              │
│   disputeMarket() → Disputed                                     │
│   redeemWinnings() → payout = bet × multiplier / SCALE           │
└──────────────────────────────────────────────────────────────────┘
                                        ▲
                                        │ Contract Events
                                        ▼
                   ┌────────────────────────────────────────┐
                   │            BACKEND (Node.js)            │
                   │────────────────────────────────────────│
                   │ Listens to events                      │
                   │ Stores data in PostgreSQL              │
                   │ API for frontend                       │
                   │ Auto tasks (cron)                      │
                   └───────────────────┬────────────────────┘
                                       │ SQL
                                       ▼
                      ┌──────────────────────────────────┐
                      │        DATABASE (PostgreSQL)      │
                      │──────────────────────────────────│
                      │ markets, bets, transactions       │
                      └──────────────────────────────────┘

┌──────────────────────────────────────────────────────────┐
│                       DEVELOPMENT                         │
│──────────────────────────────────────────────────────────│
│ Hardhat: Compile, Test, Deploy, ABI, Scripts             │
│ Networks: BNB Testnet / Localhost                        │
└──────────────────────────────────────────────────────────┘


```
